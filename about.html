Diabetic Retinopathy Severity Grading
from Fundus Photographs

Abstract

Diabetic Retinopathy (DR) is a diabetes-induced retinal disorder and one of the major causes of preventable blindness worldwide. Early detection and accurate severity grading play a vital role in preventing permanent vision loss. This project presents an automated deep learning-based system to classify diabetic retinopathy severity from retinal fundus images using a hybrid convolutional neural network model.

Existing System

In the existing system, diabetic retinopathy diagnosis is performed manually by ophthalmologists through visual inspection of retinal fundus images. This process is time-consuming, subjective, and requires high expertise. Due to the increasing number of diabetic patients, manual screening is not scalable and may lead to delayed diagnosis.

Proposed System

The proposed system automates diabetic retinopathy severity grading using deep learning techniques. A hybrid CNN model is employed to extract robust retinal features and classify fundus images into five severity levels. The system is deployed as a web-based application using Flask, enabling real-time prediction through a user-friendly interface.

Dataset Description

The model is trained and evaluated using the APTOS 2019 Blindness Detection dataset. The dataset contains high-resolution retinal fundus images categorized into five classes: No DR, Mild, Moderate, Severe, and Proliferative DR.

Methodology

  1. Data collection and labeling from APTOS 2019 dataset
  2. Image preprocessing and resizing
  3. Data augmentation
  4. Hybrid CNN feature extraction
  5. Model training and evaluation

Model Architecture

EfficientNetB0 InceptionV3
  • EfficientNetB0 – fine-grained feature extraction
  • InceptionV3 – multi-scale feature learning

Advantages

  • Fast and automated DR grading
  • High accuracy and consistency
  • User-friendly interface
